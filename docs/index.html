Components

NetlifyForm

Initializes state and functions with useNetlifyForm and provides them with a context with <NetlifyFormProvider/>. Renders a form using <NetlifyFormComponent/> inside the context.

Available props:

NameRequired?Description
childrenrequiredChildren rendered inside the form like inputs.
formPropsnot requiredAdditional props for the <NetlifyFormComponent/>. Can be used to override default methods like onSubmit and onReset.
propsnot requiredPassed as arguments to useNetlifyForm for setting options for the Netlify Form.

Required props: name, action (action should be set, so that the form also works without JavaScript)

Required if using honeypot: honeypotName (it's recommended to use a honeypot)

Required if using reCAPTCHA: enableRecaptcha

useNetlifyForm

Hook used by <NetlifyForm /> to create handlers and state which are used inside the form context.

Available props:

NameRequired?Description
namerequiredName of the form.
actionrequiredForm action to fallback to when JavaScript is disabled.
honeypotNamenot requiredNeeded when a honeypot is used because SSR needs to know it in advance.
enableRecaptchanot requiredSet this if reCAPTCHA is used because SSR needs to know it in advance.
onSuccessnot requiredHandler when data has been submitted to Netlify.
onFailurenot requiredHandler when Netlify sent an error.

NetlifyFormProvider

Providing values to NetlifyFormContext.

Available props:

NameRequired?Description
childrenrequiredChildren of context.
propsrequiredContext values.

NetlifyFormContext

Form context returned by createContext.

useNetlifyFormContext

Hook-based variant for consuming <NetlifyFormContext />.

NetlifyFormComponent

Renders a form with handlers provided by NetlifyFormContext.

Available props:

NameRequired?Description
ChildrenrequiredForm content like inputs.
propsrequiredAdditional props passed to the form. Can be used to override default methods like onSubmit and onReset.

Honeypot

Component to create a honeypot field which provides extra spam prevention. If a bot fills this field, Netlify dismisses the submitted form data.

Available props:

NameRequired?Description
labelnot requiredLabel as text before honeypot field.
propsnot requiredAdditional props for input of the honeypot.

reCAPTCHA

In Netlify dashboard you have to set SITE_RECAPTCHA_KEY and SITE_RECAPTCHA_SECRET. Also, make sure to provide SITE_RECAPTCHA_KEY to the reCAPTCHA component on the client using siteKey-prop.

Available props:

NameRequired?Description
siteKeyrequiredRecaptcha public site key.
invisiblenot requiredSet whether reCAPTCHA should be invisible or not.
